# project_bab

Team F5 - PROJECT BAB

## 개요
**밥**은 먹고 **공**부하자 : **밥공**

이 프로젝트는 사용자들이 식사를 함께할 수 있는 친구를 찾는데 도움을 주는 어플리케이션을 개발하는 것을 목표로 합니다. 사용자들은 본인과 성향이 잘 맞는 친구를 찾아 식사를 함께할 수 있습니다.
